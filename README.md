# tpExpress
## Requirements 
    Have node.js and npm installed
## Install
### On Linux
In terminal: 
    * Go to the git project
    * Type in "chmod +x linux.sh"
    * Type in "./linux.sh"
    * Open the following url in your web browser : http://localhost:8080

### On windows
In terminal:
    * Go to the git project
    * Have cygwin installed
    * Type in "sh .\windows.sh"
    * Open the following url in your web browser : http://localhost:8080
